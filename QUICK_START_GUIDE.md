**How to use:**
1. Click "Create Task" button
2. Enable "Recurring Task" toggle
3. Select frequency type
4. Set interval and days (for weekly)
5. Task automatically reschedules after completion

**Two completion options:**
- **Mark Complete (âœ“)** - Completes current instance, schedules next occurrence

### 3. Subtasks
Break down complex tasks into smaller steps:

**How to use:**
1. Click "Create Task" or edit existing task
2. Type subtask in the input field
3. Click "Add Subtask" or press Enter
4. Add multiple subtasks as needed
5. Check off subtasks as you complete them

**Important:** The main task complete button is **disabled** until all subtasks are finished!

### 4. Smart Completion Logic
- **Regular tasks with subtasks**: Complete button disabled until all subtasks done
- **Recurring tasks**: Always can mark instance complete, but "Finish Whole Task" disabled until subtasks done
- Visual feedback: Disabled buttons are grayed out and non-clickable

## ğŸ“‹ Creating Your First Task

### Basic Task
1. Click the **"Create Task"** button (+ icon)
2. Enter a **title** (required)
3. Add a **description** (optional)
4. Select **category** (Work, Personal, Health, etc.)
5. Choose **priority** (Low, Medium, High)
6. Set **difficulty** (Easy, Medium, Hard)
7. Click **"Create Task"**

### Task with Subtasks
1. Follow basic task steps above
2. In the "Subtasks" section, type first subtask
3. Click "Add Subtask" or press Enter
4. Repeat for additional subtasks
5. Click "Create Task"
6. Complete subtasks by checking them off
7. Main complete button enables when all subtasks done

### Recurring Task
1. Follow basic task steps above
2. Toggle **"Recurring Task"** ON
3. Select **frequency** (Daily, Weekly, Monthly, Custom)
4. For weekly: Select specific days
5. For custom: Set interval in days
6. Click "Create Task"
7. Task appears with ğŸ”„ badge

### Task with Due Date
1. Follow basic task steps above
2. Enable **"Set Due Date"** toggle
3. Select date and time
4. Click "Create Task"
5. Overdue tasks show warning indicators
